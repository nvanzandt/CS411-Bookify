# CS411-Bookify

## About 
This is a group project that was made for my CS411: Software Engineering Class. Bookify is a web application 
that generates a Spotify playlist inspired by a user's chosen book.

## Technologies 
- React
- Flask 
- MySQL 
- Spotify Web API 
- OpenAI API
